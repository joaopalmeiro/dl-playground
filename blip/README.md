# BLIP for Image Captioning

## Development

- `conda env create -f environment.yml`
- `conda activate blip-dev`
- `git clone https://github.com/salesforce/LAVIS.git`
- `cd LAVIS`
- `pip install .`
- `cd ..`
- `conda list`
- `python script.py`
- `isort --profile black script.py && black script.py`
